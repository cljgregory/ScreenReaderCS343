Specifications

For this tutorial you will need:

  • Windows 10/11
  • Google Chrome
  • Windows Narrator

    • What is Windows Narrator?

    Windows Narrator lets you use your PC without a mouse to complete common tasks if youâ€™re blind or have low vision. It reads and interacts with things on the screen, like text and buttons. You can use Narrator to read and write email, browse the internet, and work with documents.

    How To Start

    To start the Narrator, press the Windows Key + Ctrl + Enter
    If you wish to change any Narrator settings, press the Windows Key + Ctrl + N

    Windows Narrator Hot Keys

      Navagating the Page

    • Move through apps and web pages: Tab or Arrow keys
    • Selecting and interacting: Enter
    • Stop Narrator from reading: Ctrl

      • Volume Control

    • Increase Narrator volume: Ctrl + plus sign (+) + numbers(to select volume)
    • Decrease Narrator volume: Ctrl + minus sign (-) + numbers(to select volume)

      • Page Navagation

    • To read the current line: I or Up arrow key
    • To read the next line: O
    • To read the previous line: U
    • To read the current page: Ctrl + I.
    • To read the next page: Ctrl + O.
    • To read the previous page: Ctrl + U
    • To read by sentence: PageUp or PageDown
      • To read previous sentence: Left arrow key
      • To read next sentence: Right arrow key
    • If you would like to know where a hyperlink will take you, press Narrator + Ctrl + D
